PHP mySQL Profiler
==================

A very basic Profiler to show mySQL query activity in a PHP application.

Usage
-----
To use the profiler in your application, just include it in your PHP file.

Example usage
-------------

    <?php
    include 'profiler.php';
    ?>

Output
------

<table>
<tr>
<td>hh:mm:ss</td>
<td>mySQL query</td>
</tr>
<tr>
<td></td>
<td>mySQL query</td>
</tr>
</table>
